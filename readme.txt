== Description ==

WP-ContentBox - это плагин обеспечивающий автоматическую синхронизацию готовых текстов из сервиса [contentBox](https://contentbox.ru) на ваш сайт.

== Installation ==

1. Зарегистрируйтесь на сайте [contentBox.ru](https://contentbox.ru/register) и добавьте новый проект и получите в настройках API ID и API Token
2. Установите плагин из каталога WordPress
3. Перейдите в настройки плагина и заполните обязательные поля: API ID и API Token
4. Выберите рубрику и статус для постов по умолчанию
